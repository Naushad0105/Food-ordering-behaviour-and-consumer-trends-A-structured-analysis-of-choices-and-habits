# Food Ordering Behaviour and Consumer Trends

## 📌 Project Overview
Food Ordering Behaviour and Consumer Trends is a data analytics project developed using Tableau Public and Flask. The project analyzes customer food ordering patterns, cuisine preferences, meal types, and city-wise trends through interactive dashboards and stories. It helps restaurants and food delivery businesses gain valuable insights for better decision-making.

---

## 🎯 Objectives
- Analyze customer food ordering behaviour.
- Identify popular cuisines and meal preferences.
- Compare ordering trends across different cities.
- Generate data-driven business insights using interactive visualizations.

---

## ✨ Features
- Interactive Tableau Dashboard
- Interactive Tableau Story
- City-wise Order Analysis
- Cuisine Preference Analysis

## 🚀 How to Run

1. Clone the repository.
2. Install the required packages.

```
pip install flask
```

3. Run the Flask application.

```
python app.py
```

4. Open your browser and visit:

```
http://127.0.0.1:5000
```
